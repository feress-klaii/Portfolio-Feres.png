## 🚀 Quick Start

### Requirements

- **Node.js** version 16 or higher
- **npm** or **yarn** package manager

### Installation Steps

1. **Clone the repository:**
   ```bash
   git clone https://github.com/feress-klaii/Portfolio-Feres.png.git
   cd Portfolio-Feres.png
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start development server:**
   ```bash
   npm run dev
   ```

   Open your browser to `http://localhost:5173` to see the site with hot-reload enabled.
